# 옛한글 자모·결합·Unicode 매핑

19세기 朝鮮 文獻에는 다음과 같은 옛한글 자소가 빈번히 등장한다. 표준 Unicode (Hangul Jamo, Hangul Jamo Extended) 매핑으로 정규화하면 modern 도구에서 정상 렌더링된다.

## 핵심 옛한글 자소

| 자소 | Unicode | 명칭 | 음가 | 비고 |
|------|---------|------|------|------|
| ᆞ | U+11A2 | 아래아 (HANGUL JUNGSEONG ARAEA) | ɐ (현대 ㅏ에 가까움, ㅗ-ㅏ 사이) | 19세기까지 빈번. 1933년 한글맞춤법 통일안에서 폐지 |
| ㆍ | U+318D | 호환 아래아 (호환 자모, 단독 표시용) | 위와 동일 | display only, 결합용은 U+11A2 사용 |
| ᇫ | U+11EB | 반치음 종성 | z (유성 마찰음) | 16세기까지 살아있다가 사라짐. 19세기엔 잔영 |
| ᅀ | U+1140 | 반치음 초성 | z | 흔치 않음 |
| ㅿ | U+317F | 호환 반치음 (display) | z | display only |

| ᄫ | U+112B | 순경음 비읍 초성 | β | 15-16세기 위주, 잔영 |

## 자주 보이는 결합 syllable

옛한글 syllable은 (초성 + 중성 + 종성) Hangul Jamo 결합으로 표현한다. 한국 표준은 Hangul Syllables 영역 (U+AC00-U+D7A3)에 미리 합성된 글자가 있지만, 옛한글은 미합성으로 자모 결합.

| 결합 | Unicode (Jamo) | 음가 | 흔한 등장 패턴 |
|------|---------------|------|-----------|
| ᄒᆞ | U+1112 + U+11A2 | hɐ ≈ 하 | 'ᄒᆞ다'(하다), 'ᄒᆞ나니라'(하나니라), 'ᄒᆞ더라'(하더라) |
| ᄂᆞ | U+1102 + U+11A2 | nɐ ≈ 나 | 'ᄒᆞᄂᆞ니라'(하나니라)에서 두 번째 syllable |
| ᄃᆞ | U+1103 + U+11A2 | tɐ ≈ 다 | |
| ᄉᆞ | U+1109 + U+11A2 | sɐ ≈ 사 | |
| ᄆᆞ | U+1106 + U+11A2 | mɐ ≈ 마 | |
| ᄋᆞ | U+110B + U+11A2 | ɐ ≈ 아 | |
| ᄒᆞᆯ | U+1112 + U+11A2 + U+11AF | hɐl ≈ 할 | |
| ᄒᆞᆫ | U+1112 + U+11A2 + U+11AB | hɐn ≈ 한 | |
| ᄒᆞ외 | U+1112 + U+11A2 + U+11D0(ᇁ) | hɐwe | 옛 문서 종결 |

## 정규화 정책

옛한글 → modern hangul 변환 시 일반적 정책:

1. **ᆞ (아래아) → ㅏ**: 'ᄒᆞ' → '하'. 19-20세기 정서법 변동 따라 가장 흔한 매핑.
2. **ᆞ (아래아) → ㅗ**: 일부 어형 (ᄆᆞ → 모, ᄉᆞᆷ → 솜)에서 ㅗ로 변천. 어휘별 검토 필요.
3. **ᇫ → ㅅ 또는 ø**: 반치음 → ㅅ (ᄉᆞᆷ < *ᇫᆞᆷ → 살)이거나 탈락. 어휘별.
4. **ᄫ → ㅂ 또는 ø**: 순경음 비읍 → ㅂ (또는 탈락)
5. **연철·중철 분철 정상화**: 옛 문서는 연철 (`이라` → `ㅣ + 라` 또는 `이ㄹ + 아`) 표기 빈번. 정규화 시 분철 (`이라`).

본 스킬의 매핑 테이블에는:
- `value`: 표준 Unicode Jamo 결합형 (예: `"ᄒᆞ"` = U+1112 + U+11A2)
- `modern`: 현대 한글 (예: `"하"`)
- 둘 다 기록하여 원전 보존 + 검색 편의 양립

- **HCI-Acacia, *명조** 등의 폰트: HanyangPUA 채택. 본 스킬은 이들 폰트 PUA를 표준 Jamo로 매핑.

## hypua 라이브러리 — 자동 변환 ⭐

**`reference/hypua_table.csv`** (5660건, kiwiyou/hypua, Unlicense, public domain) 한양 PUA → IPF(첫가끝) Unicode jamo 매핑 테이블이 본 스킬에 포함된다. **옛한글 PUA의 거의 모든 codepoint를 자동 변환**한다. 시각 판독 시도하지 말고 이 테이블을 1순위로 사용하라.

```python
# 사용 예시
mapping = {}
for line in open('reference/hypua_table.csv', encoding='utf-8'):
    pua_hex, jamo_seq = line.strip().split(',', 1)
    cp = int(pua_hex, 16)
    jamo = ''.join(chr(int(x,16)) for x in jamo_seq.split('&'))
    mapping[cp] = jamo

# 예: U+F537 → 'ᄒᆞ' (ᄒ + 아래아)
print(mapping[0xF537])  # ᄒᆞ
```

`extract_pua.py`는 이 테이블을 자동 로드하여 옛한글 PUA를 자동 매핑한다.

## 본 스킬에서 hypua로 식별된 사례

- **최식2011 (2026-05-09 작업)**: 18개 옛한글 PUA를 hypua가 100% 자동 정확 매핑
  - U+F537 (TT70xx 다수) — `ᄒᆞ` (137회+) — 任圭直 본문 동사 어간 '하-'
  - U+E283 — `ᄂᆞ` (단순 '나' with 아래아, 'ᄒᆞᄂᆞ니라'의 두 번째 syllable)
  - U+E3A8 — `ᄃᆡ` ('이로ᄃᆡ' 등)
  - U+F53A — `ᄒᆞᆫ`, U+F53C — `ᄒᆞᆯ`
  - U+E1AD — `ᄀᆞᆯ` (언해 'ᄀᆞᆯᄋᆞ샤ᄃᆡ' = 가라사대)
  - U+EE88 — `ᄋᆞ`, U+E97D — `ᄉᆞ`, U+E999 — `ᄉᆡ`
  - U+E38A — `ᄃᆞ`, U+E38F — `ᄃᆞᆯ`, U+E39E — `ᄃᆞᆺ`
  - U+E46C — `ᄅᆞ`, U+E47E — `ᄅᆞᆺ`
  - U+E288 — `ᄂᆞᆯ`, U+E285 — `ᄂᆞᆫ`
  - U+E0CD — `ᅟᅵᆫ`, U+E0CE — `ᅟᅵᆯ` (filler + ㅣ + 받침)
- **시각 판독의 한계 확인**: 18개 중 시각으로 정확히 식별한 것은 6개(33%). 나머지는 미세한 점·획 차이로 다른 자모. **hypua 우선 사용 필수**.
